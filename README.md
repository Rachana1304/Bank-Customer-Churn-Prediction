Bank Customer Churn Prediction & Risk Scoring
 Project Overview
Customer churn is one of the major challenges faced by banks, as losing existing customers directly impacts revenue and long-term profitability.
This project uses Machine Learning to predict customers who are likely to leave the bank and assigns a risk score to each customer. An interactive Power BI dashboard was also developed to help business stakeholders monitor churn trends and make data-driven decisions.
Objectives
Predict customer churn using Machine Learning.
Compare multiple classification models.
Generate customer risk scores.
Identify key factors driving churn.
Build an interactive Power BI dashboard.
Provide business recommendations.
Dataset
The dataset contains 10,000 customer records with demographic, financial, and behavioral information.
Key features include:

Geography
Gender
Age
Balance

Exploratory Data Analysis
Key findings:
Germany recorded the highest churn rate.
Inactive customers were more likely to churn.
Customers with one banking product had higher churn.
Older customers showed greater churn tendencies.
Credit card ownership had minimal influence on churn.
Machine Learning Models
Three classification algorithms were evaluated:
Logistic Regression
Decision Tree
Random Forest
Random Forest achieved the best predictive performance and was selected as the final model.
